Gazpacho

July 26, 2014

This was pretty delicious, and it made a ton. The only tweak I'd make it less onion and more cucumber. It calls for a whole red onion, and the one I used was enormous, so it came out way too oniony. I saved it with some plain yogurt, but next time I think I'll cut the onion in half and add another cucumber.
